소개 기술

웹 퍼블리싱은 사용자 중심의 웹사이트 제작에 초점을 맞추는 것입니다.
이를 위해, 사용자 경험과 인터페이스에 대한 깊은 이해를 바탕으로, 사용자 친화적인 웹사이트를 개발해야 합니다. 뿐만 아니라, 웹 표준과 웹 접근성을 준수하여, 모든 사용자가 웹사이트를 편리하게 사용할 수 있도록 합니다.

저는 디자인을 시작으로 퍼블리싱까지 많은 것을 배워 왔습니다. 그럼에도, UI 공부를 하다보면 매번 새롭고 끝이 없다는 것을 느낍니다. 내일은 어떤 기술이 나올지 기대됩니다.
